# CodeGig

> Simple Crud Operation for user and posts. This app uses Node, Express, Sequalize (Postgres)

## Quick Start

``` bash
# Install dependencies
npm install

# Serve on localhost:5000
npm start
# Dev Server (Nodemon)
npm run dev
```
